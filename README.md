# Open URL In Browser

This project is to create a Chromium-supported extension that allows the user to open an URL in a different browser making it easy to migrate open websites.

The features should include:

1. Add target browsers: e.g. Safari, Chrome, Brave, Helium, etc
2. Set default browser to target
3. Open current URL in other browser
4. Open all tabs in active window in browser
5. Open selected tabs in other browser
6. Close tabs moving to new browser
7. Add recovery ability if error while trying to open tabs in other browser
  a. This can be a menu item in the extension popup
